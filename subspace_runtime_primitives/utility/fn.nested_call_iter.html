Expand description

Returns an interator over call, and any calls nested within it.

The iterator yields all calls in depth-first order, including calls which contain other calls. Ignored calls (such as pallet_utility::Call::__Ignore) are yielded themsevles, but their contents are not.

This function doesn’t use stack recursion, so there’s no need to check the recursion depth.
