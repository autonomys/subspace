fn maybe_nested_utility_calls(&self) -> Option<Vec<&RuntimeCallFor<Runtime>>>

If this call is a pallet_utility::Call<Runtime> call, returns the inner RuntimeCall.

Runtimes can override this default implementation if they want to ignore (or not ignore) certain utility calls. For example, a stack limit check shouldn’t ignore Call::__Ignore.
