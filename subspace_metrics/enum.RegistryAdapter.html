Enum RegistryAdapter 

Source 
pub enum RegistryAdapter {
    PrometheusClient(Registry),
    Substrate(Registry),
    Both(Registry, Registry),
}
Expand description

Metrics registry adapter for prometheus-client and Substrate frameworks. It specifies which metrics registry or registries are in use.

Variants§

§

PrometheusClient(Registry)

Uses only the prometheus-client metrics registry.

§

Substrate(Registry)

Uses only the Substrate metrics registry.

§

Both(Registry, Registry)

We use both Substrate and prometheus-client metrics registries.
